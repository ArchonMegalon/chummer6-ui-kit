# Chummer UI Kit - Change Plan

Date: 2026-03-09

## Mission

Own tokens, themes, shell chrome, layout primitives, feedback badges, accessibility helpers, and cross-head UI building blocks.

## Guardrails

- No domain DTOs.
- No HTTP clients.
- No local storage.
- No rules math.

## First milestones

- token canon and preview/gallery
- Blazor and Avalonia adapters
- shared shell, card, list, banner, chip, stale-state, and accessibility primitives
- package adoption in presentation and play
